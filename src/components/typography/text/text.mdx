# Typography

Our typography components should be used when rendering any kind of text. We divide them in 4 groups: _Headline_, _Subheadline_, _Detail_, _Body_.

## Usage

```jsx
import { Text } from '@commercetools-frontend/ui-kit'
```

## Headline

<div style={{ border: '1px solid magenta', padding: 16 }}>

<Text.Headline elementType="h1">This is a H1</Text.Headline>

<Text.Headline elementType="h2">This is a H2</Text.Headline>

<Text.Headline elementType="h3">This is a H3</Text.Headline>

</div>

```jsx

```
